# D1

[Cloudflare API](https://developers.cloudflare.com/api/resources/d1/subresources/database/) for D1 databases written in Crystal.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     d1:
       github: joseafga/d1
   ```

2. Run `shards install`
